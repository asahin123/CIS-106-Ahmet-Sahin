This program asks the user to select US or Metric conversion and input a given distance. 
Then the program converts the given distance to miles and displays the result.

Main Program
    Get Miles
    Get Choice
    Either:
        Calculate yards
        Calculate feet
        Calculate inches
        Display US
    Or:
        Calculate kilometers
        Calculate meters
        Calculate centimeters
        Display metric

Get Choice
    Parameters:
        None
    Process:
        Display prompt
        Get choice of US or Metric distance conversion value (US, metric)
    Return Value:
        Choice

Calculate Kilometers
    Parameters:
        Miles
    Process:
        Convert miles to kilometers
    Return value:
        kilometers

Calculate Meters
    Parameters:
        Miles
    Process:
        Convert miles to meters
    Return value:
        meters

Calculate Centimeters
    Parameters:
        Miles
    Process:
        Convert miles to centimeters
    Return value:
        centimeters
        
Calculate Yards
    Parameters:
        Miles
    Process:
        Convert miles to yards
    Return value:
        yards
        
Calculate Feet
    Parameters:
        Miles
    Process:
        Convert miles to feet
    Return value:
        feet
        
Calculate Inches
    Parameters:
        Miles
    Process:
        Convert miles to inches
    Return value:
        inches

Display US
    Parameters:
        miles
        yards
        feet
        inches
    Process:    
        Display miles, yards, feet, inches
    Return Value:
        None

Display Result
    Parameters:
        miles
        kilometers
        meters
        centimeters
    Process:    
        Display miles, kilometers, meters, centimeters        
    Return Value:
        None
