This program asks the user for a single line of text containing a 
first name and last name and print out the name in the form last name, 
first initial, such as Lastname, F.

Main Program
    Get Full Name
    Process Full Name
    Display Full Name

Get Full Name
    Parameters:
        none
    Process:
        Please enter your name
    Return value:
        fullname

Process Full Name
    Parameters:
        fullname
    Process:
        Convert the full name into last name, first initial using string methods.
    Return value:
        name
        
Display Full Name
    Parameters:
        processed name
    Process:
        Display full name in the form "lastname, F.".
    Return value:
        none
